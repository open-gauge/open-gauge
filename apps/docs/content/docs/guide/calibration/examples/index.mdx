title: Worked examples
description: Four fully worked calibration examples you can replicate through the UI to cross-check Open Gauge's calculation engine.
---

Four fully worked calibration examples, each exercising a different part of Open Gauge's calibration
engine. Every number in them was produced by running the real production code against the
input data shown, then independently re-derived by hand from the underlying formulas to
confirm they match. If you enter the same asset/channel setup and the same data points through
the UI, you should see the same numbers — see
[Rounding](./rounding-and-troubleshooting.mdx) for the display
precision rules.

Background reading: [Calibration](../overview.mdx) for what each number means and why,
[Standards & glossary](../../reference/standards-glossary.mdx) for the GUM/ISO 17025 clauses
these formulas implement.
