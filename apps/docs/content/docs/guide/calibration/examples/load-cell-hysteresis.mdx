## The math

Fit (degree 1, all 12 points): $a = 0.99833$, $b = -0.76668$.

Max error = **0.863215 N**, at point 11 (reference 100 N, measured 101.8 N) → %FS error =
$0.863215/500 \times 100\% = \mathbf{0.1726\%}$.

**Hysteresis** — Open Gauge groups points by reference value and takes the largest spread of measured
values within any group where the reference value repeats with both an ascending and
descending segment present:

| Reference | Measured values at this reference | Span |
|---|---|---|
| 0 N | 0.2, 0.3, 1.0 | 0.8 |
| 100 N | 100.6, 101.8 | 1.2 |
| 200 N | 200.9, 201.9 | 1.0 |
| 300 N | 301.0, 301.7 | 0.7 |
| 400 N | 401.1, 401.6 | 0.5 |

Largest span = **1.2 N** (at 100 N) → **hysteresis = 1.2**.

**Repeatability** — only 0 N has 3+ readings (0.2, 0.3, 1.0):

$$
\bar{x} = \frac{0.2 + 0.3 + 1.0}{3} = 0.5, \qquad
s^2 = \frac{(0.2-0.5)^2 + (0.3-0.5)^2 + (1.0-0.5)^2}{3-1} = \frac{0.09+0.04+0.25}{2} = 0.19
$$

$$
\text{repeatability} = \sqrt{0.19} = 0.435890\ \text{N}
$$

**Uncertainty budget:**

| Source | u |
|---|---|
| fit_residuals (Type A, dof = 10) | 0.487526 |
| resolution (Type B) = $0.2/\sqrt{12}$ | 0.057735 |

- Combined: $u_c = \sqrt{0.487526^2 + 0.057735^2} = \mathbf{0.490933\ N}$.
- $\nu_{\text{eff}} \approx \mathbf{10.3}$.
- Coverage factor $k = \Phi^{-1}(0.975) \approx 1.96$ (from Confidence 95%, not entered).
- Expanded: $U = 0.490933 \times 1.96 = 0.962211 \approx \mathbf{0.96\ N}$ (2 sig figs).

**Decision rule — try both and compare.** The accuracy spec is ±0.15% FS, so the tolerance is
flat across the range: $\text{tolerance} = 0.0015 \times 500\ \text{N} = 0.75\ \text{N}$. The
sweep's largest error (0.863215 N, at reference 100 N) exceeds that:

| Decision rule | Check (max error = 0.863215, tolerance = 0.75) | Result |
|---|---|---|
| Simple acceptance | $0.863215 \le 0.75$? No | **DOES NOT CONFORM** |
| Shared risk (+U) | $0.863215 - 0.962211 = -0.098996 \le 0.75$ | **CONFORMS** |

This is a genuinely realistic case: a load cell whose raw error exceeds its tight ±0.15% FS
spec, but whose overall measurement uncertainty is large enough that, under a shared-risk rule,
the lab and customer agree it's still acceptable. A good test of both "real fails happen" and
"the rule you pick changes the outcome."
